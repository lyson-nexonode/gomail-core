# gomail-core

> Sovereign mail server written in Go — SMTP · IMAP · JMAP

[![CI](https://gitlab.com/lyson-nexonode/gomail-core/badges/main/pipeline.svg)](https://gitlab.com/lyson-nexonode/gomail-core)

## Stack
- **Language**: Go 1.22+
- **Storage**: MySQL 8 + Redis 7
- **Protocols**: SMTP (RFC 5321) · IMAP4rev1 (RFC 3501) · JMAP (RFC 8620)
- **Infra**: Docker · Kubernetes

## Quick start
```bash
docker compose up
```
